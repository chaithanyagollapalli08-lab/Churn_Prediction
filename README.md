<h2>🚨 Problem Statement</h2>
<p>
Telecom companies face significant revenue loss due to customer churn.
Customers often leave the service without prior notice, making it difficult for businesses to take preventive action.
</p>
<p>
This project aims to predict whether a customer will churn or stay using Machine Learning techniques.
</p>

<h2>🎯 Objective</h2>
<ul>
    <li>Analyze customer behavior patterns</li>
    <li>Identify factors influencing churn</li>
    <li>Predict whether a customer will leave or stay</li>
    <li>Help businesses improve customer retention strategies</li>
</ul>

<h2>📂 Dataset Information (Kaggle)</h2>
<ul>
    <li><b>Source:</b> Kaggle Telecom Customer Churn Dataset</li>
    <li><b>Total Rows:</b> ~7,000+ customer records</li>
    <li><b>Total Features:</b> 20+ attributes</li>
</ul>

<h3> Key Features:</h3>
<ul>
    <li>Gender</li>
    <li>Senior Citizen</li>
    <li>Tenure</li>
    <li>Monthly Charges</li>
    <li>Total Charges</li>
    <li>Internet Service</li>
    <li>Contract Type</li>
    <li>Payment Method</li>
    <li>Churn (Target Variable)</li>
</ul>

<h2>📌 Key Insights</h2>
<ul>
    <li>Customer contract type strongly influences churn behavior</li>
    <li>Higher monthly charges increase churn probability</li>
    <li>Longer tenure customers are less likely to churn</li>
    <li>Simple models performed competitively with complex models</li>
</ul>

<h2>🚀 Final Outcome</h2>
<p>
